# 🚲 K-Store Sistema

Sistema de administración para **K-Store** — Tienda online de repuestos para bicicletas.

## Módulos planeados

- 📦 **Inventario** — Control de stock de repuestos
- 🛒 **Compras** — Registro de compras a proveedores
- 💰 **Ventas** — Ventas con descuento automático de inventario
- 📊 **Reportes** — Resúmenes para el contador

## Stack tecnológico

- **Frontend**: Next.js + React
- **Backend/DB**: Supabase (PostgreSQL)
- **Deploy**: Vercel
- **Control de versiones**: GitHub

## Cómo empezar

```bash
npm install
npm run dev
```
